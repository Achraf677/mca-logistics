# UI Kit — MCA Logistics Admin

Recreation pixel-cohérente de l'admin desktop (`admin.html`) en s'appuyant
sur la palette **Asphalt & Speed Red** v3.64 (en prod sur `login.html`) et la
typographie cible (Syne + DM Sans).

- `index.html` — Dashboard admin : sidebar + topbar + KPI grid + tableau "Dernières livraisons"
- `login.html` — Écran de connexion (shell glass + ambient orbs + role switch)

> Recréations cosmétiques. Pas de logique métier (pas de Supabase, pas de
> RLS, pas d'auth réelle). Pour la logique réelle, voir le repo `Achraf677/mca-logistics`.
